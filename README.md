# Netflix Data Analytics

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix content data to uncover trends, patterns, and insights related to movies and TV shows available on Netflix.

## Objectives
- Data Cleaning and Preprocessing
- Descriptive Statistics
- Genre Analysis
- Release Year Analysis
- Content Rating Analysis
- Country-wise Content Distribution
- Correlation Analysis
- Trend Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Distribution of Movies vs TV Shows
- Most Popular Genres
- Content Growth Over Time
- Country-wise Content Production
- Rating Distribution

## Dataset
Netflix Titles Dataset
